# SUVIDHA
## Smart Urban Virtual Interactive Digital Helpdesk Assistant

A full-stack civic services platform with AI-powered assistance.

## Tech Stack
- **Frontend:** React.js + Vite + Tailwind CSS
- **Backend:** Node.js + Express.js + MongoDB Atlas
- **AI Service:** Python + FastAPI + LangChain + Gemini + FAISS

## Running the Project

### Frontend
```bash
cd client && npm run dev #till we deploy
```

### Backend
```bash
cd server && npm run dev #till we deploy
```

### AI Service
```bash
cd ai-service
venv\Scripts\activate
uvicorn app:app --reload --port 8000 #till ew learn ai and deploy
```
